## ğŸ Starting

### Using npm

```bash
# Clone this project
$ git clone https://github.com/harshalgunjal69/vdeohub-backend.git

# Access
$ cd vdeohub-backend

# Install dependencies
$ npm i

# Run the project
$ npm run dev

# The server will initialize in http://localhost:3000
```

### Using yarn

```bash
# Clone this project
$ git clone https://github.com/harshalgunjal69/vdeohub-backend.git

# Access
$ cd vdeohub-backend

# Import dependencies
$ yarn import

# Install dependencies
$ yarn install

# Run the project
$ yarn dev

# The server will initialize in http://localhost:3000
```
