<h1 id="starting"> 🏁 Starting </h1>

### Using npm

```bash
# Clone this project
$ git clone https://github.com/harshalgunjal69/Backend.git

# Access
$ cd backend

# Install dependencies
$ npm i

# Run the project
$ npm run dev

# The server will initialize in http://localhost:3000
```

### Using yarn

```bash
# Clone this project
$ git clone https://github.com/harshalgunjal69/Backend.git

# Access
$ cd backend

# Install dependencies
$ yarn

# Run the project
$ yarn dev

# The server will initialize in http://localhost:3000
```
